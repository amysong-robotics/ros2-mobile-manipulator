# 🤖 ROS2-IntelliBot 

![ROS 2](https://img.shields.io/badge/ROS_2-Humble-3498DB?logo=ros)
![Ubuntu](https://img.shields.io/badge/Ubuntu-22.04-E95420?logo=ubuntu)
![License](https://img.shields.io/badge/License-Apache_2.0-green.svg)

An advanced, open-source autonomous mobile manipulator built on ROS 2. This project integrates mapping, navigation, robotic arm manipulation, and multi-modal human-robot interaction (Voice & Gesture control) into a single mobile platform.

## 🌟 Key Features

- 🗺️ **SLAM & Mapping:** High-precision 2D/3D map generation using Cartographer / SLAM Toolbox.
- 🧭 **Autonomous Navigation:** Reliable point-to-point navigation and dynamic obstacle avoidance powered by the Nav2 framework.
- 🦾 **Object Manipulation:** Kinematics calculation and trajectory planning for the robotic arm using MoveIt 2 to perform precise grasping tasks.
- 🗣️ **Voice Control:** Natural language processing interface allowing users to control the robot's movement and actions via voice commands.
- ✋ **Gesture Recognition:** Touchless human-robot interaction using computer vision (e.g., MediaPipe) to control the robot via hand gestures.
